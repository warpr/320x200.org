releases

This page contains all bliss group releases (please visit the member page for individual member releases).

All the files available here contain an executable + data you can run out of the box on any recent Microsoft Windows (DirectX 7 or newer required).

The .zip files also contain the full source for you to compile on Linux (if you try other unices, let me know what problems come up --warp), GGI is required. (it seems SDL is included with more and more distributions, so I guess I should provide SDL versions some day :)

get bob: bob-0.1.20.zip (236 kb), manual, pouët

Bob is warp's 32k game compo entry for Mekka & Symposium 2001, it got 7th place.

Bob is a 2 player pc re-implementation of a classic c64 game called jawbreaker. It was coded by warp in C on an old pentium 133MHZ, the code is a lot cleaner and better optimized compared to 8bit :).


warp used drug powerup on avoozl.

The music was made by tharsis and the player is a hybrid of fxmp-0.0.21 and fxmp-0.0.22-pre2 with some size optimizations. fxmp by spectre of flare.

Linux/GGI port was mostly done by smoke of ecfh.

For more detailed credits and other info read the manual, which is in the .zip file and has been put online too for your convenience.
