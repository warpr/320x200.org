Welcome to the releases page. All these files contain an executable + data you can run out of the box on any recent Microsoft Windows (2000 or 95/98/ME), also contained in the .zip file are complete sources for you to compile on Linux (other unices probably won't work, please send me feedback on that).

 

bob download: bob-0.1.20.zip (235 kb), screenshot, manual

 

Bob is warp's 32k game compo entry for Mekka & Symposium 2001, it got 7th place.

Bob is a 2 player pc re-implementation of a classic c64 game called jawbreaker. It was coded by warp in C on an old pentium 133MHZ, the code is a lot cleaner and better optimized compared to 8bit :).

Music is by tharsis and the player is a hybrid of fxmp-0.0.21 and fxmp-0.0.22-pre2 with some size optimizations. fxmp by spectre of flare.

Linux/GGI port was mostly done by smoke of ecfh.

For more detailed credits and other info read the manual, which is in the .zip file and has been put online too for your convenience.

8bit download: Bliss-8bit.zip (284 kb), screenshots: 1, 2,

 

We got 3rd place in the Bizarre 2000 intro compo with 8bit. 8bit was mostly coded by warp in C++. The code is not optimized at all and the intro thus requires a 500MHZ cpu to run smooth. Sorry about that :)

Music is by tharsis and player is a slightly modified version of fxmp-0.0.20 by spectre of flare.
